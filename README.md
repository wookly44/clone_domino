## 📌 구현 기능

- [x] 메뉴 클릭 시 서브 메뉴가 나타나는 드롭다운 기능 구현
- [x] 메뉴 클릭 시 화살표 아이콘이 회전하는 애니메이션 인터랙션 구현
- [x] Swiper 라이브러리를 활용한 메인 슬라이드 구현
- [x] JavaScript를 활용한 탭 메뉴 기능 구현

- [x] 팝업 UI 및 팝업 열기/닫기 기능 구현

## 🌱 배운 점 / 느낀 점

- 메뉴 인터랙션과 탭 기능을 구현하면서 사용자 이벤트에 따라 UI가 동적으로 변경되는 구조를 이해
- FAQ 아코디언 기능과 팝업 기능 등을 구현하며 JavaScript를 활용한 UI 인터랙션 구현 경험
- Swiper 라이브러리를 활용한 슬라이드 UI 구현 경험

---

## ⚠ 아쉬운 점 / 개선하고 싶은 부분
